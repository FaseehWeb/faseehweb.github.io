Dot Dash (Beta)

This game tests your mouse control skills by challenging you to click on randomly appearing dots as quickly as possible. Once you've clicked all the dots, it calculates and displays your average reaction time.

What's in the Latest Update

Bug Fixes

  1. Adjusted the alignment of the restart button for a centered appearance.
  2. The application no longer calculates the response time for the first click.
  3. Response time is no longer calculated from the moment the restart button is clicked.

Improvements

  1. Added a feature to restart the game by clicking the spacebar while the game is running.
  2. The program now displays response time and average response time with only three significant figures, rather than the full float value.
